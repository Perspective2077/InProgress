# Border Gateway Protocol
Large routers on the internet learn routes automatically from other routers using routing protocols.
BGP lets routers tell each other things like:
```
"I can reach these IP ranges"
```

So routers exchange this information and build their routing tables dynamically.

## 4. Manual Static Routes

Sometimes administrators manually add routes.
```
10.0.0.0 → send to router 192.168.1.2
```
These are called static routes.


# How the Internet Works at Large Scale
Internet backbone routers exchange routes using
Border Gateway Protocol.

This protocol basically maps the entire internet's reachable IP ranges between networks.
